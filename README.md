# Публичные наборы правил

Каталог создаётся автоматически из исходников в `rules/`. Не редактируйте его вручную.

Здесь допустимы только домены, IP/CIDR-диапазоны, индекс и безопасный пример подключения удалённых наборов правил. Рабочие конфигурации и данные доступа должны храниться только локально.

Доступные артефакты:

- `rulesets/eu-only.json`
- `rulesets/tr-only.json`
- `rulesets/auto-vps.json`
- `txt/eu-only.txt`
- `txt/tr-only.txt`
- `txt/auto-vps.txt`
- `metadata/build-info.json`
- `metadata/domains-index.json`
- `examples/sing-box-remote-ruleset-snippet.json`

Фрагмент из `examples/` предназначен для включения удалённых наборов правил в локальную конфигурацию.
